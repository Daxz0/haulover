# About
A **haulover** (hah get it like an overhaul) of Minecraft's farming systems, slowly one bit at a time!

# Features
- Farmland no longer wets itself... You have to use watering cans!
- Added two new watering cans -> Basic & Advanced
- Crops no longer can be bonemealed! (new uses for bonemeal coming eventually)
- Watering cans can be refilled by looking at water
- And you can water crops by splashing on them with watering cans!
